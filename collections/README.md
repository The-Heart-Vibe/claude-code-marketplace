# Collections

Kuratorskie kolekcje skilli pogrupowane wg roli / fazy pracy. **Nie są to bundle pluginy** — każdy skill instalujesz osobno z jego źródła.

Cel kolekcji: pomóc zespołowi znaleźć właściwy skill dla konkretnej fazy pracy bez przeszukiwania setek dostępnych.

## Dostępne kolekcje

| Kolekcja | Dla kogo | Faz |
|----------|----------|-----|
| [Venture Builder](venture-builder.md) | Analitycy + konsultanci w obszarze venture building (The Heart edition) | 7 |
| [Onboarding VB](onboarding-vb.md) | Nowy w zespole — 20-min przewodnik krok po kroku | — |
| [Demo Prompts](demo-prompts.md) | Live demo z managerem/teammate — 10 progressive promptów, ~60-90 min | 5 |

## Jak dodać nową kolekcję

1. Stwórz `collections/<nazwa>.md` z sekcjami per faza
2. Każdy skill: nazwa, źródło (marketplace), install command, kiedy używać, przykład prompta
3. Dodaj wiersz do tabeli wyżej
4. PR
